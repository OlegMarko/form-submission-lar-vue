## Install app

`git clone git@github.com:OlegMarko/form-submission-lar-vue.git`

`cd form-submission-lar-vue`

`cp .env.example .env`

`composer install`, `php artisan key:generate`

`npm i`, `npm run watch`, `php artisan serve`
